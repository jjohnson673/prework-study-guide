HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The id is a way to identify elements and select them
  • h1, h2, h3 can all be used to distinguish individual headers
  • ul creates an unordered list of items
  • li represents in the same manner of a bullet point

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull: fetch and merge and commits from the tracking remote branch
  • git add -A: indicates adding ALL of the modifications in the working branch to the staging area
  • git push: create a new remote branch or add new changes to an existing branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function's name is followed by parentheses ()
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • The strict equality operator (===) checks to see if two values are equal, and returns a Boolean result true if the values are equal and false if the values are not equal.
